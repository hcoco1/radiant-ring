### Copy IPv4

- server Vitual Machine (VM) = Hostname: EC2AMAZ-L3OOUG8
- desktop Vitual Machine (VM) = Hostname: DESKTOP-2




1. **On the server VM**, run on PowerShell `ipconfig` and **copy the IP** from the IPv4 Address.

### Modify DNS

2. **On the desktop-2 machine** (not server), paste the IP in the DNS field of network settings and click OK.



![alt text](https://github.com/hcoco1/career-2/blob/main/images/step_1_2.png?raw=true)

---

### Rename PC

3. In **System** click on "Rename this PC (Advance), locate Computer Name Tab and  click on "Change". You can change the Computer Name (Optionally) to "administrator" and fill out the domain name "contoso.com". Click on OK
